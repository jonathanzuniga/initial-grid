# Initial Grid

> A fully customizable CSS flexbox grid system for a fast starting point.

## Usage

### Columns of the same size

```html
<div class="row">
	<div class="column">...</div>
	<div class="column">...</div>
	<div class="column">...</div>
</div>
```

### Columns of different size

```html
<div class="row">
	<div class="column-4">...</div>
	<div class="column-8">...</div>
</div>
```
